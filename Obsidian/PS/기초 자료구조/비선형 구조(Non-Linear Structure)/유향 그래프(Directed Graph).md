### 정의
- 그래프의 연결 상태를 표시하기 위해 순서쌍을 이용한 형태
- 무향 그래프와 달리 간선에 방향이 있어 두 노드 사이의 단방향적인 관계를 표현할 수 있다

⇒ 유향 그래프가 아닌것을 무향 그래프라 한다

### 구현
```cpp
#include <vector>

// T는 자료형, MAX는 노드의 개수 + 1
vector<T> g[MAX];

// n개의 간선에 대한 입력을 받음
for (int i = 0; i < n; i++) {
	// u는 시작노드 v는 도착노드
	// g[u]에 v를 넣는다는 것은 u에서 v로 갈 수 있음을 의미한다
	int u, v; cin >> u >> v;
	g[u].push_back(v);
}
```
